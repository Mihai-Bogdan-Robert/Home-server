# IT Company Server

Perfect for small IT teams and business networks.

## Services Included

- **Tailscale** - Secure VPN mesh network for remote access
- **Docker + Portainer** - Container management and deployment
- **Samba** - File sharing and network storage (SMB/CIFS)
- **Nginx** - Reverse proxy and web server
- **Netdata** - System monitoring and analytics

## Quick Start

```bash
sudo bash install.sh
```

Select services for your company needs:
- Tailscale (secure remote access)
- Docker (containerized apps)
- Portainer (container UI management)
- Samba (network file shares)
- Nginx (web/proxy)

## Hardware Requirements

- **Minimum**: 4 CPU cores, 8GB RAM, 500GB storage
- **Recommended**: 8 CPU cores, 16GB RAM, 1TB+ storage
- **For heavy load**: 16+ CPU cores, 32GB+ RAM

## Use Cases

- Secure remote access for dispersed teams
- Centralized file storage and sharing
- Container-based app deployment
- Team collaboration tools
- Network monitoring

## Security Notes

- Tailscale provides zero-trust networking
- Firewall rules should be configured
- Regular backups recommended
- Consider HTTPS certificates for web services
